# 梅花易数技能

> "梅花易数，心易通神。" —— 宋代邵雍

梅花易数是宋代易学大师邵雍（邵康节）所创的占卜方法，以数字、时间、方位等起卦，无需铜钱，灵活便捷。本技能提供完整的梅花易数排盘和自动化断卦功能。

## ✨ 核心功能

- ✨ 数字起卦（用户输入任意数字）
- ✨ 时间起卦（年月日时起卦）
- ✨ 方位起卦（按方位起卦）
- ✨ 声音起卦（按声音次数）
- ✨ 颜色起卦（按颜色对应数）
- ✨ 本卦、互卦、变卦排布
- ✨ 体用分析（体卦、用卦）
- ✨ 五行生克分析
- ✨ 自动化断卦（吉凶判断、断语）
- ✨ 应期推算
- ✨ 纯 Python 实现，无外部依赖

## 🔧 技术特点

- 🔧 多种起卦方式（数字/时间/方位/声音/颜色）
- 🔧 完整互卦、变卦算法
- 🔧 体用生克分析
- 🔧 纯 Python 实现，无外部依赖
- 🔧 支持 JSON 格式输出
- 🔧 命令行界面友好

## 📖 快速开始

### 安装

```bash
# 克隆仓库
git clone https://github.com/dglijin-oss/meihua-yishu-skill.git
cd meihua-yishu-skill

# 确保 Python 3.8+ 已安装
python3 --version

# 赋予执行权限
chmod +x scripts/meihua_pan.py
```

### 基本使用

```bash
# 数字起卦（输入 2 个数字）
python3 scripts/meihua_pan.py --method number --numbers "7,8"

# 时间起卦（当前时间）
python3 scripts/meihua_pan.py --method time

# 指定时间起卦
python3 scripts/meihua_pan.py --method time --date "2026-04-10 15:00"

# 方位起卦（东/南/西/北/东南/西南/西北/东北）
python3 scripts/meihua_pan.py --method direction --direction 南

# 指定问事类型（带断卦分析）
python3 scripts/meihua_pan.py --method number --numbers "7,8" --question 财运

# 输出 JSON 格式
python3 scripts/meihua_pan.py --method number --numbers "7,8" --json
```

### 命令行参数

| 参数 | 简写 | 说明 | 默认值 |
|------|------|------|--------|
| --method | -m | 起卦方法 (number/time/direction) | number |
| --numbers | -n | 2 个数字 (1-99) | 可选 |
| --date | -d | 日期时间 (YYYY-MM-DD HH:MM) | 当前时间 |
| --direction | -dir | 方位 (东/南/西/北/东南/西南/西北/东北) | 可选 |
| --question | -q | 问事类型 | 通用 |
| --json | -j | 输出 JSON 格式 | 否 |

## 📚 梅花易数基础知识

### 八卦基础

| 卦名 | 符号 | 五行 | 自然 | 数字 |
|------|------|------|------|------|
| 乾 | ☰ | 金 | 天 | 1, 6 |
| 兑 | ☱ | 金 | 泽 | 2, 7 |
| 离 | ☲ | 火 | 火 | 3, 9 |
| 震 | ☳ | 木 | 雷 | 4 |
| 巽 | ☴ | 木 | 风 | 5 |
| 坎 | ☵ | 水 | 水 | 1, 6 |
| 艮 | ☶ | 土 | 山 | 7 |
| 坤 | ☷ | 土 | 地 | 8 |

### 起卦方法

#### 数字起卦
- 第一个数 ÷ 8 取余数 = 上卦
- 第二个数 ÷ 8 取余数 = 下卦
- 两数之和 ÷ 6 取余数 = 动爻

#### 时间起卦
- 年 + 月 + 日 之和 ÷ 8 取余数 = 上卦
- 年 + 月 + 日 + 时 之和 ÷ 8 取余数 = 下卦
- 年 + 月 + 日 + 时 之和 ÷ 6 取余数 = 动爻

#### 方位起卦
- 按方位对应数字起卦

### 体用关系

| 关系 | 含义 | 吉凶 |
|------|------|------|
| 用生体 | 事来生我 | 大吉 |
| 体生用 | 我去生事 | 小吉（泄气） |
| 体克用 | 我能克事 | 吉（需努力） |
| 用克体 | 事来克我 | 凶 |
| 体用比和 | 比和相助 | 吉 |

### 互卦推导

- 互卦由本卦的二、三、四爻组成下卦，三、四、五爻组成上卦
- 互卦代表事情的发展过程

### 变卦推导

- 变卦由本卦的动爻变化而来
- 阳爻变阴爻，阴爻变阳爻
- 变卦代表事情的最终结果
